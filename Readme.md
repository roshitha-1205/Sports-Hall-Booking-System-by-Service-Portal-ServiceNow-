# 🏟️ Sports Hall Booking System

The **Sports Hall Booking System** is a custom-built **ServiceNow** application developed using **Service Portal** to simplify and automate sports facility reservations. It enables users to view real-time slot availability, submit booking requests, and track approvals through an intuitive portal interface. The system automates approvals, notifications, and status updates using **Flow Designer**, **Business Rules**, and **Scheduled Jobs**, ensuring a secure, efficient, and scalable booking experience.

---

## 📌 Overview

The Sports Hall Booking System digitizes the entire sports facility reservation process by replacing manual booking methods with an automated workflow. It improves user convenience, enhances operational efficiency, and provides administrators with complete control over sports hall management.

## 📖 Instructions

1. Log in to the ServiceNow instance using valid user credentials.
2. Open the **Sports Hall Booking Service Portal**.
3. Select the desired sports hall, booking date, and available time slot.
4. Review the available slots and booking charges displayed on the portal.
5. Complete the required booking details and submit the booking request.
6. Track the booking status from the portal:
   - ⏳ Pending
   - ✅ Approved
   - 🎉 Confirmed
   - ❌ Rejected
7. Approvers receive an email notification to approve or reject the request.
8. Once approved, the system automatically:
   - Confirms the booking
   - Updates the booking status
   - Sends a confirmation notification to the user
9. If the booking is cancelled, applicable cancellation charges are calculated automatically.
10. Administrators can manage sports halls, time slots, approvals, reports, and configurations through the ServiceNow backend.
