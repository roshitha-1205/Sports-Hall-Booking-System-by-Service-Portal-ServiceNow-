
## Creation & Implementation of Scheduled Jobs Script

### Activity 1: Navigation of Scheduled Jobs

- Navigate to the **All** menu.
- Search for **Scheduled Jobs**.
- Under **System Definition**, select **Scheduled Jobs**.
- Open the Scheduled Jobs list.
- Click the **New** button to create a new Scheduled Job.

- Open the newly created Scheduled Job.
- Navigate to the **Advanced** tab.

- Implement the script according to the project logic.
