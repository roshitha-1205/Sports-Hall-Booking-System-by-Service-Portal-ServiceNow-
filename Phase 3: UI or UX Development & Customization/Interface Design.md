
## Creation of Service Portal

### Activity 1: Creation of Service Portal

1. Navigate to the **Application Navigator**.
2. Click on **All**.
3. Search for **Service Portal**.
4. Under **Service Portal**, select **Portals**.
5. Click on the **New** button.
6. Fill in the portal details:

| Field | Value |
|--------|-------|
| **Title** | Sports Zone |
| **URL Suffix** | `sz` |
| **Home Page** | Select or create a new homepage (`sports_booking`) |
| **Theme** | Default Theme → **La Jolla** |

7. Click **Submit** or **Save** to create the portal.
